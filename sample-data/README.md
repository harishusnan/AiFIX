# Sample Data

This folder contains synthetic sample financial data used for testing the AiFIX application.

- You may upload the CSV file (or your own CSV file) in Data Vault
- For financial reference materials, you can upload the PDF file (or your own reference) in Knowledge Base

Data Vault Usage:
1. Open the AiFIX app
2. Go to Data Vault
3. Upload `your_financial_file.csv`
4. View results in Analytics page

Knowledge Base Usage:
1. Open the AiFIX app
2. Go to Knowledge Base
3. Upload `your_financial_reference_file.pdf`
4. Chat with AiFIX on the user query bar (top right of the AiFIX app)
